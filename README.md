В проекте предоставлены коды для курсовой работы.

В файле DataGenerators.java предоставлена программа для генерации тестовых данных для задач.
В файле Knapsack01.java предоставлена программа для решения задачи 0/1 Knapsack.
В файле LongestCommonSubsequence.java предоставлена программа для решения задачи Longest Common Subsequence.
В файле MatricChainMultiplication.java предоставлена программа для решения задачи Matric Chain Multiplication.

Программы для решения задач предоставляют решения той или иной задачи тремя методами: без оптимизации, с оптимизацией через мемоизацию, с оптимизацией через табуляцию.
